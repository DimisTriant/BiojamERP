# BiojamERP – Ομαδική Εργασία Java

Αυτό το repository περιέχει την ομαδική εργασία που αναπτύχθηκε στα πλαίσια του μαθήματος Π3, με αντικείμενο την υλοποίηση ενός συστήματος ERP για μια επιχείρηση μαρμελάδων.

## 📦 Περιεχόμενα

- `ΟΜΑΔΑ_19_Π3.zip`: Περιλαμβάνει όλο τον πηγαίο κώδικα σε Java, την τεκμηρίωση (PDF) και διαγράμματα (Visual Paradigm).
- Αναλυτική έκθεση ~80 σελίδων.
- UML διαγράμματα (Use Case, Class, Sequence κ.ά.)

## 💡 Σκοπός

Η εργασία προσομοιώνει ένα βασικό ERP σύστημα με λειτουργίες όπως:
- Καταχώρηση παραγγελιών
- Διαχείριση αποθεμάτων
- Παραγωγή αναφορών

## 🛠 Τεχνολογίες

- Java (console application)
- Visual Paradigm για UML σχεδίαση
- ZIP αρχείο για πλήρη παρουσίαση

## 👥 Ομάδα

Ομαδικό έργο στο πλαίσιο προγραμματιστικού μαθήματος.
